## 📌 Project Overview
This project is part of the Digital Literacy course. It focuses on building awareness and practical skills required to effectively and responsibly use digital technologies in academic and professional environments.

## 🟢 Task 1: Digital Literacy Awareness Infographic

### 📖 Description
For this task, I created a one-page infographic to explain the concept of digital literacy and its importance for students. The infographic covers key topics such as digital tools, safe internet practices, professional online presence, and email etiquette.

### 🛠️ Tool Used
Canva (online design tool)

### 🎯 Key Highlights
- Defined digital literacy in simple terms
- Listed useful tools like Google Docs and GitHub
- Included safety practices for internet use
- Explained professional online behavior
- Added basic email etiquette guidelines
